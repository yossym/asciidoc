== 演算子

.四則演算
[option="header"]
|====
|演算子|説明
|+|加算
|-|減算
|*|乗算
|/|除算
|%|剰余
|**|累乗
|====


[source,bash]
----
echo $((1 + 2))
3

x=100
y=100
echo $((x+y))
200
----
